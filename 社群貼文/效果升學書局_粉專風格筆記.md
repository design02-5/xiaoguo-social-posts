# 效果升學書局 粉專寫文風格筆記

> 依 2026-07 提供的 4 篇實際貼文分析，作為後續自動排程寫文的風格依據。

## 近期實際發布公式（2026-07-14 起，優先於下方舊版分析）
拿 7/2、7/3 兩篇（Claude 起草、使用者發布前大改過）的貼文一比對，發現使用者實際發布的版本，跟舊版分析（反問句開場＋顧問語氣敘事）不一樣，改用更直接的公式：

- 開頭三件套：大標句直接當第一句（不寫情境鋪陳／不用反問句）→ 一句正向宣言 → 立刻放當期 CTA 連結，格式固定為 `👉 [當期推薦頁連結]建立屬於你的學習策略`
  - **這個連結每兩個月會換一次**（對應當期推薦書單頁或活動頁），**寫文前一定要跟使用者確認當下這兩個月的連結是什麼**
- 中段固定三年級分類：📘國小／📗國中／📙高中，每類 2-3 句情境說明＋該類別實際商品短連結（例如 lihi1.me 短網址），連結必須依當次主打商品分流，不能三類都導去同一個首頁
- 結尾用正向宣言句，不用反問句收尾（例：「讓開學後的準備更有方向」而非「你想帶著什麼樣的自己開學？」）
- Hashtag 混「品牌固定詞＋具體品類搜尋詞」（例：#國小參考書 #國中參考書 #弱科加強），可以把原本草稿大標的slogan整句直接變成一個長 hashtag 保留（例：#這一次還有機會把結果拉回來），但不放進本文

## 舊版分析（2026-07 前 4 篇基準，僅供參考，非目前優先公式）

## 品牌定位（最優先原則）
- **效果不是一般賣書的書店，而是給學生建議的角色。**
- 寫文時先站在「幫學生解決準備升學的困擾」出發，書籍/連結是「建議的解方」，不是推銷的商品。
- 避免「這本書很好、快來買」這種賣貨語氣；改用「你現在這個階段適合這樣準備」的顧問語氣，書單只是佐證建議的工具。
- 導購連結、結尾模板還是要放（品牌露出、導流），但主文的敘事重心是「建議」而非「銷售」。

## 開頭鉤子
- 常用「反問句」或「情境共鳴句」開場，先戳中學生/家長焦慮或痛點
  - 例：「不知道該買什麼書、怎麼準備？」
  - 例：「很多學生不是英文不好，而是不知道如何把想法寫成一篇完整作文。」
- 開頭常搭配情緒/主題 emoji（🎓 🎯 ✨），一則一顆，不濫用

## 結構
- 大主題下切成 2～3 個「路徑/階段」小節，每小節用 emoji 當 icon 引導（📊 📝 🎯 🏫）
- 小節內用「➡」或「>>」接推薦連結，格式固定：`挑選OO參考書➡ 連結`
- 商品清單用「‧」條列，格式：`‧書名重點 >> 連結`
- 段落極短，善用換行，手機閱讀不用橫向滾動

## 結尾固定模板（每篇幾乎不變，直接沿用）
```

【效果網路書店】全國最大、最齊全升學參考書店
📱下載效果升學書局 APP

🛒效果網路書店：https://www.efficient.com.tw/
📗官方客服LINE：https://lin.ee/sXrWE6v
```

## Hashtag 慣例
- 品牌固定 tag：#效果升學書局 #效果網路書店
- 加 3～6 個主題 tag，依內容換（例：#學測準備 #高中參考書 #會考參考書 #暑假書單）

## 互動引導
- 偶爾在文末邀請留言（例：留「年級＋科目＋準備狀況」抽神秘小禮物），拉互動率
- 不是每篇都要有，重大主題/長文適合加

## 語氣
- 鼓勵、同理學生壓力，但目的明確導向「選書/買書」
- 會用比喻軟化說教感（例：「學習就像一場長途旅行...」）
- 專業但不生硬，像懂升學制度的朋友在給建議，不是制式廣告詞

## 常用素材類型
- 依年級/階段分（國中／高中、各科目）
- 常搭配活動主題：學測、會考、分科測驗、暑假衝刺、108課綱素養題

## CSV 主題關鍵字欄位填寫規則（2026-07-21 起）

排程規劃表（`效果升學書局_排程規劃表.csv`）的「寫文方向」欄位，填寫方式從「完整標語句」改成「主題關鍵字」：

- 舊寫法（不再使用）：`④（暑假起跑）👉暑假不是放空，是重建學習節奏的開始`
- 新寫法：只填主題關鍵字，例如 `暑假讀書計畫`、`會考倒數衝刺`、`高中選書迷思`
- Routine 執行時會依這個關鍵字，結合下方「自動網路搜尋規則」查到的即時資訊，自己發想切角與完整文案，不需要使用者事先想好句子
- 「產品/素材」欄位、「備註」欄位用法不變

## 自動網路搜尋規則（2026-07-21 起）

Routine 執行當天，依當天主題關鍵字做以下兩類搜尋，取得即時資訊後才動筆：

1. **教育部／測驗中心官方公告**：查證跟主題相關的考期、報名期限等時效性資訊是否有異動，避免貼文寫出過期或錯誤的日期
2. **效果自家網站／粉專最新活動**：確認當期主打書單頁、CTA連結（`👉 [當期推薦頁連結]建立屬於你的學習策略`，每兩個月換一次）是現在還有效的版本，以及各年級分類（📘國小／📗國中／📙高中）目前應該導去哪個商品短連結

### 搜尋不到明確資訊時的退場寫法
- 不可自行編造日期、報名期限或連結
- 改用不掛時效性的通用句子帶過該段（例如把「113學年會考將於OO月OO日登場」改成「會考將近，這是準備方向的最後衝刺期」這種不寫死日期的說法）
- 不能因為搜尋不到資訊就整篇不產出，只影響用到該資訊的那個段落

## 來源內部檢核區塊格式（2026-07-21 起）

每篇輸出的 HTML，在貼文本文結束後（結尾固定模板之後），新增一個「內部審核用」區塊，格式如下：

```html
<div class="internal-sources" style="margin-top:40px;padding-top:16px;border-top:1px dashed #ccc;font-size:12px;color:#999;">
  <strong>【內部審核用，勿貼到FB】本篇引用來源：</strong>
  <ul>
    <li>[來源說明] <a href="[來源網址]">[來源網址]</a></li>
  </ul>
</div>
```

- 只列出「這次搜尋且實際寫進文案」的來源，不用列出所有搜尋過但沒用到的結果
- 若這篇完全沒用到任何搜尋來的即時資訊（例如純粹用通用寫法），這個區塊可以省略整段不輸出
- 這個區塊放在貼文本文之後、結尾模板之後，使用者複製貼文本文到 FB 時不會連這段一起複製到

## 配圖規則（2026-07-22 起，半自動版）

Routine **不呼叫任何圖像生成 API**，只生成一段配圖提示詞，交給使用者自己用 ChatGPT/DALL·E 手動生圖。

### 提示詞內容規則
提示詞需包含以下設計規則（來自 CORE_RULES.md）：
- 這是全新生成的圖，不涉及既有素材裁切問題，但畫面需重新構圖排版，不是把文字硬套上去
- 風格：高級感、真實感、專業感、電影感
- 禁止：背景雜物、假感、不自然光線
- 依當天主題關鍵字決定畫面意象（例如「暑假讀書計畫」可以是專注讀書情境、書桌/書本意象），避免出現看得出來是AI生成的違和細節

### 提示詞呈現位置
提示詞寫進「內部審核用」區塊裡（比照來源清單的呈現方式），格式：

```html
<div class="internal-image-prompt" style="margin-top:12px;font-size:12px;color:#999;">
  <strong>【內部審核用，勿貼到FB】配圖提示詞：</strong>
  <p>[提示詞內容]</p>
  <p>請把生成好的圖存成 <code>[固定檔名]</code>，上傳到 repo 同一個資料夾</p>
</div>
```

### 固定檔名規則
圖檔與同一篇 HTML 用同樣的日期/序號命名，副檔名 `.png`：
- HTML：`社群貼文/YYYY-MM-DD_效果升學書局.html`（或 `_1.html`/`_2.html`）
- 圖檔：`社群貼文/YYYY-MM-DD_效果升學書局.png`（或 `_1.png`/`_2.png`）

### 貼文本文的 <img> 標籤
不管圖檔當下存不存在，貼文本文最前面一律要先插入：

```html
<img src="./YYYY-MM-DD_效果升學書局.png" alt="配圖" style="max-width:100%;">
```

（檔名對應當篇的固定檔名）圖檔還沒上傳前這裡會顯示破圖，使用者依上面提示詞生圖、存成指定檔名上傳到 repo 同資料夾後就會自動正確顯示，**不需要回頭再編輯這份 HTML**。
